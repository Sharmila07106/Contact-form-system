# Contact Form System (Flask)

## Features
- Contact form
- Admin panel on the same page
- No database
- No email

## How to run
1. Download the ZIP file from this repository
2. Unzip it
3. Install Python if not installed
4. Install Flask:
   pip install flask
5. Open terminal inside the project folder
6. Run:
   python app.py
7. Open browser:
   http://127.0.0.1:5000/
